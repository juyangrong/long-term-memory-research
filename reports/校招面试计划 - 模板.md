### 针对性面试问题

#### 1. 计算机基础 (12 分钟)

**数据结构与算法：**
```
Q1. [根据简历技术栈选择]
    例如：HashMap 的原理是什么？
    参考答案要点：
    - 数组 + 链表/红黑树
    - 哈希函数
    - 冲突解决（链地址法）
    - 时间复杂度 O(1)
    评分：⬜优秀 ⬜良好 ⬜一般 ⬜较差

Q2. [算法题]
    例如：手写快速排序
    参考答案要点：
    - 分治思想
    - 基准选择
    - 递归实现
    - 时间复杂度 O(nlogn)
    评分：⬜优秀 ⬜良好 ⬜一般 ⬜较差

Q3. [根据项目经历追问]
    例如：你的项目中用到了什么数据结构？为什么选择它？
    评分：⬜优秀 ⬜良好 ⬜一般 ⬜较差
```

**计算机网络：**
```
Q4. TCP 三次握手和四次挥手？
    参考答案要点：
    - 三次握手建立连接（SYN, SYN+ACK, ACK）
    - 四次挥手断开连接（FIN, ACK, FIN, ACK）
    - 为什么是三次不是两次？（防止失效的连接请求）
    评分：⬜优秀 ⬜良好 ⬜一般 ⬜较差

Q5. HTTP 和 HTTPS 的区别？
    参考答案要点：
    - HTTPS = HTTP + SSL/TLS
    - 端口不同（80 vs 443）
    - 安全性（加密传输）
    - 证书验证
    评分：⬜优秀 ⬜良好 ⬜一般 ⬜较差
```

**操作系统：**
```
Q6. 进程和线程的区别？
    参考答案要点：
    - 进程是资源分配单位，线程是调度单位
    - 进程间独立，线程共享进程资源
    - 切换开销：进程 > 线程
    评分：⬜优秀 ⬜良好 ⬜一般 ⬜较差
```
