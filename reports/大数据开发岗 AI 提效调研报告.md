**适用场景：**

**AI 生成示例：**
```python
# 自然语言描述
"从 MySQL 同步用户数据到 Hive，每天增量同步，
需要：1. 去重  2. 敏感字段加密  3. 分区存储"

# AI 生成完整 ETL 代码
def sync_user_data():
    # 读取增量数据
    source_data = read_mysql_incremental(
        table='users',
        last_sync_time=last_sync
    )
    
    # 去重处理
    deduped_data = source_data.drop_duplicates(subset=['user_id'])
    
    # 敏感字段加密
    deduped_data['phone'] = encrypt_phone(deduped_data['phone'])
    deduped_data['id_card'] = encrypt_id_card(deduped_data['id_card'])
    
    # 写入 Hive 分区表
    write_to_hive(
        data=deduped_data,
        table='ods.users',
        partition=f"dt={today}"
    )
```

**示例：**
```yaml
# AI 生成的数据质量规则
rules:
```
