## 隐私保护技术

### 1. 差分隐私 (Differential Privacy)

**原理**: 在数据中添加噪声，使攻击者无法确定特定个体的信息

**实现**:
```python
import numpy as np

def add_dp_noise(data: np.array, epsilon: float = 1.0) -> np.array:
    """添加差分隐私噪声"""
    sensitivity = 1.0  # 敏感度
    scale = sensitivity / epsilon
    noise = np.random.laplace(0, scale, data.shape)
    return data + noise

# 使用示例
original_embedding = np.array([0.1, 0.5, 0.3, ...])
private_embedding = add_dp_noise(original_embedding, epsilon=0.5)
```

**优缺点**:
| 优点 | 缺点 |
|------|------|
| ✅ 数学可证明的隐私保护 | ⚠️ 降低检索准确性 |
| ✅ 易于实现 | ⚠️ 需要调整隐私预算 (epsilon) |
| ✅ 与现有系统兼容 | ⚠️ 不适用于小规模数据 |

**适用场景**:
- ✅ 大规模用户记忆
- ✅ 统计分析场景
- ⚠️ 不适用于精确检索

### 2. 同态加密 (Homomorphic Encryption)

**原理**: 在加密数据上直接进行计算，无需解密

**实现**:
```python
from tenseal import ts

# 创建上下文
context = ts.context(
    ts.SCHEME_TYPE.CKKS,
    poly_modulus_degree=8192,
    coeff_mod_bit_sizes=[60, 40, 40, 60]
)
context.generate_galois_keys()
context.generate_relin_keys()

# 加密向量
vector = [0.1, 0.5, 0.3, 0.8]
encrypted_vector = ts.ckks_vector(context, vector)

# 加密状态下计算相似度
encrypted_similar = encrypted_vector.dot(encrypted_vector)
```

**优缺点**:
| 优点 | 缺点 |
|------|------|
| ✅ 完全隐私保护 | ⚠️ 计算开销大 (+300%) |
| ✅ 支持向量检索 | ⚠️ 实现复杂 |
| ✅ 无需信任服务器 | ⚠️ 仍在研究阶段 |

**适用场景**:
- ✅ 高敏感数据 (医疗/金融)
- ⚠️ 性能要求不高的场景

### 4. 敏感信息过滤

**实现**:
```python
# sensitive_filter.py
import re
from typing import List, Dict, Tuple, Optional

class SensitiveFilter:
    PATTERNS = {
        "id_card": r"\b\d{17}[\dXx]\b",  # 身份证号
        "phone": r"\b1[3-9]\d{9}\b",     # 手机号
        "bank_card": r"\b\d{16,19}\b",   # 银行卡号
        "email": r"\b[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Z|a-z]{2,}\b",
        "password": r"(?i)password|passwd|pwd",
        "address": r"(?i)地址 | 住址 | 街道 | 小区",
    }
    
    SENSITIVITY_LEVELS = {
        "high": ["id_card", "bank_card", "password", "health"],
        "medium": ["phone", "email", "address"],
        "low": ["preference", "habit"]
    }
    
    def __init__(self, action: str = "mask"):
        """
        action: "mask" (脱敏) | "reject" (拒绝) | "log" (仅记录)
        """
        self.action = action
    
    def filter(self, text: str) -> Tuple[Optional[str], List[Dict]]:
        """过滤敏感信息"""
        detected = []
        
        for name, pattern in self.PATTERNS.items():
            matches = re.findall(pattern, text)
            if matches:
                detected.append({
                    "type": name,
                    "count": len(matches),
                    "level": self._get_level(name)
                })
        
        # 检查是否有高敏感信息
        high_sensitivity = any(d["level"] == "high" for d in detected)
        
        if high_sensitivity and self.action == "reject":
            return None, detected
        
        if detected:
            if self.action == "mask":
                return self._mask(text), detected
            elif self.action == "log":
                return text, detected
        
        return text, []
    
    def _get_level(self, pattern_name: str) -> str:
        for level, patterns in self.SENSITIVITY_LEVELS.items():
            if pattern_name in patterns:
                return level
        return "low"
    
    def _mask(self, text: str) -> str:
        """脱敏处理"""
        for name, pattern in self.PATTERNS.items():
            if name == "password":
                continue  # 密码直接拒绝
            text = re.sub(pattern, "[REDACTED]", text)
        return text

# 使用示例
filter = SensitiveFilter(action="mask")
text = "我的手机号是 13800138000"
filtered, detected = filter.filter(text)
print(f"过滤后：{filtered}")  # 我的手机号是 [REDACTED]
print(f"检测到：{detected}")
```

### GDPR (欧盟通用数据保护条例)

**核心要求**:
1. ✅ 明确告知数据收集目的
2. ✅ 用户同意 (可撤回)
3. ✅ 数据访问权
4. ✅ 数据导出权
5. ✅ 被遗忘权 (删除权)
6. ✅ 数据可携带权

**实现清单**:
```python
# GDPR 合规实现
class GDPRCompliance:
    def __init__(self, memory_service):
        self.memory = memory_service
    
    def export_data(self, user_id: str) -> Dict:
        """数据导出"""
        return {
            "memories": self.memory.get_all_memories(user_id),
            "access_logs": self._get_access_logs(user_id),
            "export_date": datetime.now().isoformat()
        }
    
    def delete_data(self, user_id: str, confirmation: str):
        """数据删除 (被遗忘权)"""
        if confirmation != "I_CONFIRM_DELETION":
            raise ValueError("需要明确确认")
        
        # 逻辑删除
        self.memory.soft_delete_all(user_id)
        
        # 物理删除 (后台任务)
        self._schedule_hard_delete(user_id)
        
        # 审计日志
        self._log_deletion(user_id)
    
    def get_access_logs(self, user_id: str) -> List:
        """访问日志"""
        return self._query_logs(user_id)
```

### 日志格式

```json
{
  "timestamp": "2026-03-14T10:30:00Z",
  "event_id": "evt_xxx",
  "user_id": "user_123",
  "session_id": "session_456",
  "action": "memory_write",
  "resource": {
    "type": "memory",
    "id": "mem_789"
  },
  "details": {
    "content_hash": "sha256:...",
    "sensitivity_level": "medium",
    "retention_policy": "1_year"
  },
  "actor": {
    "type": "user",
    "id": "user_123",
    "ip": "192.168.1.100"
  },
```

### 审计内容

| 事件类型 | 记录内容 | 保留期 |
|----------|----------|--------|
| 记忆创建 | 用户 ID、内容哈希、敏感度 | 2 年 |
| 记忆检索 | 用户 ID、查询哈希、结果数 | 1 年 |
| 记忆更新 | 用户 ID、旧/新哈希 | 2 年 |
| 记忆删除 | 用户 ID、删除确认 | 7 年 |
| 数据导出 | 用户 ID、导出内容类型 | 7 年 |
| 访问失败 | 用户 ID、失败原因、IP | 90 天 |

### 实现示例

```python
# audit_logger.py
import json
import hashlib
from datetime import datetime
from typing import Dict, Optional

class AuditLogger:
    def __init__(self, log_path: str = "audit.log"):
        self.log_path = log_path
    
    def log(self, event: Dict):
        """记录审计事件"""
        event["timestamp"] = datetime.now().isoformat()
        event["event_id"] = self._generate_id()
        
        with open(self.log_path, "a") as f:
            f.write(json.dumps(event) + "\n")
    
    def log_memory_operation(self, action: str, user_id: str, 
                            memory_id: str, content: str = None,
                            sensitivity: str = "low"):
        """记录记忆操作"""
        event = {
            "action": f"memory_{action}",
            "user_id": user_id,
            "resource": {
                "type": "memory",
                "id": memory_id
            },
            "details": {
                "content_hash": hashlib.sha256(content.encode()).hexdigest() if content else None,
                "sensitivity_level": sensitivity
            }
        }
        self.log(event)
    
    def _generate_id(self) -> str:
        import uuid
        return f"evt_{uuid.uuid4().hex}"

# 使用示例
auditor = AuditLogger()
auditor.log_memory_operation("write", "user_123", "mem_456", 
                            "用户偏好信息", "medium")
```
