# 01-关键术语表

**最后更新**: 2026-03-14

---

## A

### Archival Memory (归档记忆)
长期存储的记忆，通常使用向量数据库存储，支持语义检索。与核心记忆和对话记忆共同构成分层记忆架构。

### Attention Mechanism (注意力机制)
Transformer 模型的核心组件，用于计算输入序列中不同位置的重要性权重。在原生记忆中，可通过调整注意力实现记忆增强。

---

## B

### Biologically-Inspired Memory (生物启发式记忆)
模仿人脑记忆机制的 AI 记忆系统，包括情景记忆、语义记忆、程序性记忆的分工，以及记忆巩固、遗忘曲线等生物特性。

---

## C

### Consolidation (记忆巩固)
将短期记忆转化为长期记忆的过程。在 AI 系统中，通常指将对话记忆摘要并存储到归档记忆的过程。

### Core Memory (核心记忆)
分层记忆架构中最稳定的一层，存储用户身份、核心偏好等不易变化的信息。通常限制在 2000-3000 tokens。

### Cypher
Neo4j 图数据库的查询语言，用于查询和更新图结构数据。

---

## D

### Differential Privacy (差分隐私)
在数据中添加噪声以保护个人隐私的技术。在记忆系统中，可用于保护用户记忆不被精确还原。

### Dual-Process Theory (双系统理论)
卡尼曼提出的认知理论，将思维分为 System 1 (快思考) 和 System 2 (慢思考)。在记忆系统中，可用于设计快慢检索路径。

---

## E

### Embedding (向量嵌入)
将文本、图像等数据转换为高维向量的过程。向量之间的距离表示语义相似度。

### Embedding Model (嵌入模型)
用于生成向量嵌入的模型，如 text-embedding-3-small、bge-m3、m3e-base 等。

### Ebbinghaus Forgetting Curve (艾宾浩斯遗忘曲线)
描述记忆随时间遗忘的曲线。在 AI 记忆系统中，可用于实现记忆强度随时间衰减。

---

## F

### Federated Learning (联邦学习)
一种分布式机器学习方法，数据保留在本地，只上传模型梯度。可用于保护用户记忆隐私。

---

## G

### GDPR (通用数据保护条例)
欧盟的数据保护法规，赋予用户访问、导出、删除个人数据的权利 (被遗忘权)。

### Graph Database (图数据库)
使用图结构 (节点 + 边) 存储数据的数据库，如 Neo4j、Memgraph、NebulaGraph。适合存储实体关系。

### Graph Memory (图记忆)
使用图结构存储的记忆，支持复杂的关系推理和多跳查询。

---

## H

### Hierarchical Memory (分层记忆)
将记忆分为多个层级 (核心/归档/对话) 的架构，类似计算机的存储层次结构。

### Homomorphic Encryption (同态加密)
允许在加密数据上直接进行计算的技术。可用于在保护隐私的前提下进行向量检索。

---

## I

### In-Context Memory (原生记忆)
不依赖外部存储，通过优化模型内部机制 (注意力、参数) 实现的记忆能力。

---

## K

### Knowledge Graph (知识图谱)
使用图结构表示知识的系统，节点表示实体，边表示关系。

---

## L

### LangChain
用于构建 LLM 应用的 Python 框架，提供记忆管理、工具调用等功能。

### Letta (原 MemGPT)
分层记忆管理的先驱框架，支持自主记忆整理。

---

## M

### Mamba
基于状态空间模型 (SSM) 的序列模型，具有线性复杂度，适合长序列处理。

### Mem0
开源的 LLM 记忆层框架，支持向量记忆和图记忆 (Mem0g)。

### MemGPT
Letta 的前身，提出分层记忆架构的研究项目。

### Memory Layer (记忆层)
位于 LLM 和应用之间的中间层，负责记忆的存储、检索、更新。

### Multi-Hop Query (多跳查询)
在图数据库中，通过多条边连接的路径查询。例如"找出我朋友的朋友"。

---

## N

### Neo4j
流行的图数据库，使用 Cypher 查询语言。

### NER (命名实体识别)
从文本中识别和分类实体 (人名、地名、组织等) 的技术。

---

## P

### Pinecone
云托管的向量数据库服务，免运维。

### Procedural Memory (程序性记忆)
存储技能和习惯的记忆类型，如"如何骑自行车"。

---

## Q

### Qdrant
开源的向量数据库，支持自托管和云服务。

---

## R

### RAG (Retrieval-Augmented Generation)
检索增强生成，从外部知识库检索信息并注入 LLM 上下文的技术。

### RWKV
结合 RNN 和 Transformer 优点的序列模型，适合长序列处理。

---

## S

### Semantic Memory (语义记忆)
存储抽象知识和概念的记忆类型，如"巴黎是法国首都"。

### Sleep Replay (睡眠回放)
生物启发式记忆中的机制，在离线时重放重要记忆以强化巩固。

### SSM (State Space Model)
状态空间模型，一种序列建模范式，Mamba 基于此。

---

## T

### Top-K Retrieval (Top-K 检索)
从数据库中检索最相似的 K 个结果。

---

## V

### Vector Database (向量数据库)
专门用于存储和检索向量数据的数据库，如 Qdrant、Pinecone、Milvus。

### Vector Memory (向量记忆)
使用向量数据库存储的记忆，支持语义相似度检索。

---

## W

### Working Memory (工作记忆)
短期存储和处理信息的记忆系统，对应 AI 中的对话记忆/上下文窗口。
