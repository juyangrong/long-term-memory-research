## 1. 技术概述

### 1.1 核心原理

向量记忆是将文本、图像等信息转换为高维向量 (Embedding)，存储在向量数据库中，通过语义相似度进行检索的记忆技术。

```
文本 → [Embedding 模型] → 向量 (768 维) → [向量数据库] → 相似度检索
```

#### 核心代码

```python
from mem0 import Memory
from qdrant_client import QdrantClient

# 1. 初始化向量数据库客户端
qdrant_client = QdrantClient(url="http://localhost:6333")

# 2. 初始化记忆系统
memory = Memory(
    vector_store={
        "provider": "qdrant",
        "config": {
            "client": qdrant_client,
            "collection_name": "user_memories",
            "embedding_model_dims": 768
        }
    },
    embedder={
        "provider": "huggingface",
        "config": {
            "model": "BAAI/bge-m3"
        }
    },
    llm={
        "provider": "ollama",
        "config": {
            "model": "llama3.1:8b"
        }
    }
)

# 3. 添加记忆
def add_memory(user_id: str, content: str, metadata: dict = None):
    """添加用户记忆"""
    result = memory.add(content, user_id=user_id, metadata=metadata)
    return result

# 4. 检索记忆
def search_memories(user_id: str, query: str, limit: int = 5):
    """检索相关记忆"""
    results = memory.search(query, user_id=user_id, limit=limit)
    return results

# 5. 删除记忆
def delete_memory(user_id: str, memory_id: str):
    """删除指定记忆"""
    memory.delete(memory_id, user_id=user_id)

# 6. 导出记忆 (GDPR)
def export_memories(user_id: str):
    """导出用户所有记忆"""
    return memory.get_all(user_id=user_id)

# 使用示例
if __name__ == "__main__":
    # 添加记忆
    add_memory("user_123", "用户喜欢冰美式咖啡", {"category": "preference"})
    add_memory("user_123", "用户居住在北京", {"category": "fact"})
    
    # 检索记忆
    results = search_memories("user_123", "咖啡偏好")
    for r in results:
        print(f"记忆：{r['memory']}, 得分：{r['score']}")
```

#### 高级功能
```python
# 批量添加记忆
def batch_add_memories(user_id: str, memories: list):
    """批量添加记忆"""
    results = []
    for mem in memories:
        result = memory.add(mem["content"], user_id=user_id, metadata=mem.get("metadata"))
        results.append(result)
    return results

# 带时间衰减的检索
def search_with_decay(user_id: str, query: str, decay_factor: float = 0.95):
    """带时间衰减的检索"""
    results = memory.search(query, user_id=user_id)
    
    import time
    current_time = time.time()
    
    for r in results:
        # 计算时间衰减
        age_days = (current_time - r["created_at"]) / 86400
        decay_weight = decay_factor ** age_days
        r["weighted_score"] = r["score"] * decay_weight
    
    # 按加权分数排序
    results.sort(key=lambda x: x["weighted_score"], reverse=True)
    return results

# 记忆冲突检测
def detect_conflict(user_id: str, new_content: str, threshold: float = 0.8):
    """检测新记忆是否与现有记忆冲突"""
    # 检索相似记忆
    similar = memory.search(new_content, user_id=user_id, limit=5)
    
    conflicts = []
    for mem in similar:
        if mem["score"] > threshold:
            # 高相似度但内容不同 → 可能冲突
            if mem["memory"] != new_content:
                conflicts.append({
                    "existing": mem["memory"],
                    "new": new_content,
                    "similarity": mem["score"]
                })
    
    return conflicts
```

### 4.1 检索优化

```python
# 1. 混合检索 (向量 + 关键词)
from qdrant_client.http.models import Filter, FieldCondition, MatchValue

def hybrid_search(user_id: str, query: str, vector: list, keyword: str = None):
    """混合检索：向量相似度 + 关键词过滤"""
    filter_conditions = [
        FieldCondition(key="user_id", match=MatchValue(value=user_id))
    ]
    
    if keyword:
        filter_conditions.append(
            FieldCondition(key="tags", match=MatchValue(value=keyword))
        )
    
    results = qdrant_client.search(
        collection_name="memories",
        query_vector=vector,
        query_filter=Filter(must=filter_conditions),
        limit=10
    )
    
    return results
```

## 5. 最佳实践

### 5.1 记忆分块策略

```python
def chunk_memory(content: str, max_length: int = 500, overlap: int = 50):
    """将长文本分块存储"""
    chunks = []
    start = 0
    
    while start < len(content):
        end = start + max_length
        chunk = content[start:end]
        chunks.append(chunk)
        start = end - overlap
    
    return chunks

# 使用示例
long_text = "用户详细偏好描述..." * 100
chunks = chunk_memory(long_text)
for chunk in chunks:
    add_memory("user_123", chunk)
```

### 5.3 敏感度分级

```python
SENSITIVITY_RULES = {
    "high": ["身份证号", "银行卡", "密码", "健康状况"],
    "medium": ["地址", "电话", "邮箱", "工作单位"],
    "low": ["偏好", "兴趣", "习惯"]
}

def classify_sensitivity(content: str) -> str:
    """分类敏感度"""
    for level, keywords in SENSITIVITY_RULES.items():
        if any(kw in content for kw in keywords):
            return level
    return "low"
```

## 6. 常见问题

### Q1: 向量检索不准确怎么办？
**A**: 
1. 检查 Embedding 模型是否适合你的语言
2. 调整相似度阈值 (默认 0.6-0.7)
3. 使用混合检索 (向量 + 关键词)
4. 添加重排序 (Rerank) 步骤

### Q2: 记忆太多导致检索变慢？
**A**:
1. 按用户 ID 分片存储
2. 添加类别过滤 (只检索相关类别)
3. 使用缓存 (Redis)
4. 定期归档旧记忆

### Q3: 如何处理记忆冲突？
**A**:
1. 检测冲突 (相似度 > 0.8 但内容不同)
2. 时间优先 (新记忆覆盖旧记忆)
3. 用户确认 (关键信息)
4. 版本控制 (保留历史)
