```python
def retry_with_backoff(func, max_retries=3, base_delay=1):
    for i in range(max_retries):
        try:
            return func()
        except Exception as e:
            if i == max_retries - 1:
                raise
            delay = base_delay * (2 ** i)
            time.sleep(delay)
```

**问题**: 这段代码实现什么功能？有什么优缺点？

**期望回答**:
- 指数退避重试
- 优点：避免雪崩，给系统恢复时间
- 缺点：固定异常类型，无抖动

**评分标准**: 5/3/1 分
